# 我的 GitHub Pages 网站

这是一个使用 GitHub Pages 部署的静态网站示例仓库。

## 📂 仓库内容

- `index.html` ：网站首页
- `style.css` ：网站样式
- `script.js` ：网站交互脚本

## 🚀 使用方法

1. 将仓库克隆到本地或直接编辑。
2. 在 GitHub 仓库设置中启用 **GitHub Pages**。
3. 访问 `https://username.github.io` 查看网站效果（将 `username` 替换为你的 GitHub 用户名）。

## ✨ 功能

- 基础页面布局
- 样式美化
- 按钮点击弹窗示例

## 📌 说明

这是一个最基础的 GitHub Pages 示例，适合新手快速搭建个人网站或项目主页。
